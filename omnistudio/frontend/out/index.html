Import du scénario

Téléchargez votre fichier pour commencer. Formats acceptés : Excel, CSV, Word, PDF, Markdown ou texte brut.

Pas de fichier ?

Vous pouvez saisir vos segments manuellement dans l'onglet Préparation, sans importer de fichier.

Colonne A = identifiant de l'étape (1, 2, 3...). Colonne B = texte à prononcer. La première ligne (en-tête) est ignorée. Vous pouvez choisir l'onglet à importer.

Même structure que Excel : colonne 1 = identifiant, colonne 2 = texte. Séparateur auto-détecté (virgule, point-virgule, tabulation). Première ligne ignorée. Encodage UTF-8 ou Latin-1.

Chaque étape commence par ### Étape N suivi du texte à prononcer.

Chaque paragraphe de texte = une étape. Les titres (Titre 1, Titre 2...) sont ignorés. Les tableaux et images sont ignorés. Les paragraphes longs (plus de 100 mots) sont découpés automatiquement par phrases.

Le texte est extrait automatiquement page par page. Chaque paragraphe = une étape, numérotation automatique. Les PDF scannés (images) ne sont pas supportés : le fichier doit contenir du texte sélectionnable.

Chaque paragraphe (séparé par une ligne vide) = une étape. La numérotation est automatique. Les paragraphes longs (plus de 100 mots) sont découpés automatiquement. Encodage UTF-8 ou Latin-1.

Préparation des textes

L'IA prépare vos textes pour une lecture optimale par la synthèse vocale.

Le fichier contient 3 sections, toutes optionnelles :

  • patterns — remplacements par expression régulière (sigles, abréviations)
  • parentheses — suppressions ou remplacements exacts de parenthèses parasites
  • majuscules — normalisation des mots en MAJUSCULES non naturelles
{
  "patterns": {
    "\\bEPS\\b": "éducation physique et sportive",
    "\\bRGPD\\b": "règlement général sur la protection des données"
  },
  "parentheses": {
    "(cf. annexe)": "",
    "(voir diapo)": ""
  },
  "majuscules": {
    "COMPÉTENCES": "compétences",
    "ÉVALUATION": "évaluation"
  }
}

Les patterns utilisent des expressions régulières (\b = limite de mot). Les parentheses et majuscules sont des remplacements exacts. Laissez la valeur vide "" pour supprimer.
